# Synthetic CSF image set

This directory contains deterministic PNG images generated by:

```powershell
.\.venv\Scripts\python.exe tools\generate_synthetic_images.py --output image_sets\synthetic\png
```

The set is intentionally small and local:

| Image | Purpose |
| --- | --- |
| `smooth_gradient_512x512.png` | smooth low-frequency content |
| `sharp_edges_512x512.png` | hard edges, diagonals, color bands |
| `fine_texture_512x512.png` | high-frequency texture and noise |
| `mixed_content_512x512.png` | mixed gradient, blocks and radial detail |

Use it when Kodak/CLIC downloads are unavailable, or as a quick smoke input for visual-quality metrics.
