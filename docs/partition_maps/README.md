# Partition Maps

Partition-map evidence is organized by codec.

| Path | Status |
| --- | --- |
| `vvenc/` | Generated from VVenC `D_QP` traces with validated final luma CU coordinates. |
| `vtm/` | Generated from VTM `D_QP` traces emitted by trace-enabled VTM 23.0 encoders. |

New runs write codec-separated partition maps to `docs/partition_maps/vvenc/` and `docs/partition_maps/vtm/`.
